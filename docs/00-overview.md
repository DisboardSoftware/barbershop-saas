# Overview

## Scope

This blueprint defines a runnable baseline for barbershop SaaS products, with consistent architecture, governance alignment, and local deployment.

## Included

- Frontend and backend monorepo structure
- Session auth + RBAC baseline
- PostgreSQL + Prisma persistence
- MinIO media upload pipeline
- Docker Compose local orchestration

## Non-goals

- Complete business workflows
- Production cloud deployment automation
- Full billing and observability implementation
